# 💼 Personal Portfolio

A modern and responsive personal portfolio website built with **React** and **Vite** to showcase my skills, projects, education, and experience as an aspiring Full Stack Developer.

## 🚀 About

This portfolio serves as a central place to present my work, technical skills, and projects. It is designed with a clean and responsive user interface to provide a smooth browsing experience across all devices.

## ✨ Features

- Responsive design
- Clean and modern UI
- About Me section
- Skills section
- Projects showcase
- Resume section
- Contact section
- Smooth navigation

## 🛠️ Tech Stack

- React
- Vite
- JavaScript (ES6+)
- HTML5
- CSS3 / CSS Modules
- Git & GitHub

## ⚙️ Getting Started

Clone the repository

```bash
git clone https://github.com/pokemon12-12/your-repository-name.git
```

Navigate to the project folder

```bash
cd your-repository-name
```

Install dependencies

```bash
npm install
```

Run the development server

```bash
npm run dev
```

Build for production

```bash
npm run build
```
